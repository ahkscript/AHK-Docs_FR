Pour Commencer

Avant que AutoHotkey puisse faire quoi que ce soit, il faut écrire un script lui disant quoi faire.
Pour apprendre comment faire, lire le tutoriel ou foncer directement.

Simplement : créer un fichier de script → ajouter des hotkeys → exécuter le fichier de script.

De l'aide peut aussi être demandée, ainsi que de nombreux scripts utiles trouvés, sur notre forum de discussion amicale.

Pourquoi vois-je ceci ?

AutoHotkey a été lancé sans lui dire quel fichier de script exécuter, et il n'a pas pu trouver un script dans aucun des endroits habituels.
